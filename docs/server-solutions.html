Frequently Asked Questions

Will the server come pre-configured for LLMs?

Yes! Our servers are built with a secure, pre-installed OS and basic configuration to run LLM models. We include tools to help you deploy and manage your AI workloads easily.

How do I ensure my data is secure?

We use hardware-based security tokens (mailed separately) for verification, along with encrypted OS configurations and regular firmware updates. All data stays local to your home network, avoiding cloud storage risks.

Can I run multiple LLMs on one server?

Yes! Our servers are built with high-memory configurations and scalable GPUs to support running multiple models simultaneously. For advanced setups, our 3U chassis is ideal for multi-model management.

What if I need to upgrade my server later?

Every server is designed with future-proof upgrades in mind. Our modular design allows you to swap GPUs, add more RAM, and expand storage options easily.

Is the hardware easy to set up?

We've simplified everything! The servers come with a simple web interface for setup, and our pre-installed OS includes security measures and AI tools out of the box. No technical expertise is required.

What are the power requirements?

The 2U chassis uses ~300W (varies by GPU), and the 3U chassis uses ~600W. Both are compatible with standard home power supplies, but we recommend a dedicated circuit for stability.

Do you offer customer support?

Yes! We provide 24/7 email support and documentation to help you get started.
